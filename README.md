# Project README

## Dataset
To download the dataset, follow the link provided. Once downloaded, place the dataset in the `data` folder of the repository.

## Running the Notebooks
Ensure that you have the following libraries installed:
- scikit-learn
- matplotlib
- numpy

You can install these libraries using pip:
```bash
pip install scikit-learn matplotlib numpy
```
Now you can run the Jupyter notebooks included in the repository to analyze the dataset!
